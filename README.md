# Nearest_Restaurant_Finding_Haversine_Formula
A Spring Boot REST API that finds the nearest restaurants based on a given latitude and longitude using the Haversine formula.
Restaurant location data is stored in MySQL, and the API calculates distances dynamically to return nearby results.
